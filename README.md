Это простое приложение, открывающее оверлей с веб-поиском и выбором поискового движка. Все очень просто. Вы вызываете прилажку, в первый раз запросит установить разрешение "Поверх других приложений". Если вы установили APK самостоятельно то сначала разрешите доступ к настройкам в окне "О приложении" -> Ещё (три точки вверху) -> Разрешить
